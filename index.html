git

  • 깃허브 리파지토리 설정
  • https://git-scm.com/ 다운로드 받아서 설치
  • 폴더 우클릭 git bash here 선택
  • $ git config --global user.neme "mazy"
  • $ git config --global user.email "mazy02@naver.com"
  • $ git config --list
  • 비쥬얼 스튜디오 터미널 실행
  • +클릭 후 git bash 선택
  • 터미널 master 확인 후
  • git init 초기화는 한번만
  • git add . 파일 전체 업로드
  • git status 상태확인 (안해도 됨)
  • git commit -m "first commit" 히스토리 위한 메세지 작성(같으면 오류남)
  • git remote add origin https://github.com/SciaYMY/MY-portfolio.git 로컬과 깃프로젝트 연결위해 깃허브 repository 주소 복사
  • git remote -v 잘 연결 됐는지 확인
  • git push origin master 업로드
